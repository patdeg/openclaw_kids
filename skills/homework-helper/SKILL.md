# Homework Helper — Study Tools

Productivity and study tools for school work.

## Usage

```
exec: python3 ~/skills/homework-helper/homework_helper.py pomodoro <start|stop|status>
exec: python3 ~/skills/homework-helper/homework_helper.py flashcards <topic> [--count 10]
exec: python3 ~/skills/homework-helper/homework_helper.py outline <topic>
exec: python3 ~/skills/homework-helper/homework_helper.py math <problem>
exec: python3 ~/skills/homework-helper/homework_helper.py cite <url> [--format mla|apa]
```

## Commands

- `pomodoro start` — Start a 25-min study session (5-min break after).
- `pomodoro stop` — Cancel the current session.
- `pomodoro status` — Check time remaining.
- `flashcards <topic>` — Generate study flashcards on a topic.
- `outline <topic>` — Generate an essay outline with thesis, body, conclusion.
- `math <problem>` — Solve a math problem step-by-step (show work).
- `cite <url>` — Generate a citation for a web source (MLA or APA format).

## Notes

- The pomodoro timer tracks state in workspace/pomodoro.json.
- Flashcards and outlines are generated by the AI based on the skill output.
- Math shows all work — never just the answer. The goal is learning.
- Citations use the tavily skill to extract page metadata when available.
